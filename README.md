# adscrusher

这是一个屏蔽广告插件，适用于chrome浏览器，其他浏览器厂商不清楚。

可以清除掉指定的class、id及所有iframe。处理方式：

- class: remove or display=none
- id: remove
- iframe: display=none




# 目前支持网站

- blog.csdn.net
